This project builds a machine learning model to predict whether an interaction results in a positive outcome.

I built this mainly to understand how classification models behave on real world tabular data especially when the data is messy and imbalanced.

I used a Random Forest classifier as the main model.

One important thing I explored was how changing the prediction threshold affects results:

Default threshold = high accuracy, but misses most positive cases
Lower threshold = catches more positives, but increases false positives

This trade off turned out to be more important than just improving accuracy.
